Algumas Tags de HTML

  1. ol: Utilizado para iniciar uma lista ordenada;
  2. i: Permite que o texto fique em itálico;
  3. b: Permite que o texto fique em negrito;
  4. li: Seria utilizado dentro de toda lista para indicar um list-item, ou seja, um item pertencente a uma lista;
